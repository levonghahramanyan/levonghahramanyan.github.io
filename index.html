Digital microscope on linux and windows

How to attach a Digital microscope on linux and Windows

Linux

Attaching Steps

You don't need the site

  1. Grab the microscope's USB cable
  2. Attach it to the USB port of the computer
  3. Right click under the screen
  4. Hold the crusor on "multimedia"
  5. Select "VLC media player"
  6. Select "media"
  7. Select "open capture device"
  8. In "video device" select "/dev/video0" or "/dev/video1"
  9. Click "play"

Windows

Attaching Steps

You don't need the site

  1. Grab the microscope's USB cable
  2. Attach it to the USB port of the computer
  3. Click on the Windows icon below or on the ceyboard
  4. Select "camera"
